BiblioLAB – Un ejemplo de uso de dialog

7 junio, 2017 En # CODE

bash y dialog son un buen complemento para crear formularios e instaladores para línea de comandos. Por ejemplo Debian en su modo de instalación modo texto tira de dialog para ir preguntando al usuario durante la instalación sobre raíles.
  
Es posible crear algunas otras cosas y como ejemplo sencillo BiblioLAB:
  

  
Se puede ver y bajar el código aquí: https://github.com/Ingobernable/BiblioLab
  
En los centros sociales se busca en cierto modo la autogestión y el auto-servicio aparte de promover la confianza.
  
Es interesante por tanto que cuando coges de una librería técnica de un hacklab un libro no estés molestando a otra gente para ello pero que dejes constancia de que tu te lo has llevado.
  
El motivo de dejar constancia es que otra persona que busque el mismo libro sepa al menos el correo de contacto de quien lo tiene. De ese modo se puede poner en contacto para dialogar sobre cuanto tiempo piensa tenerlo, etc…
  
Este script realizado en una mañana sirve para eso. Permite coger y dejar libros rápidamente.
  
Se ha puesto en la librería técnica del hacklab IngoberLAB 301 y puedes ir allí si quieres a coger un libro.
