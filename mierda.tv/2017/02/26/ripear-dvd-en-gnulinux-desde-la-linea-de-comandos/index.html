Ripear DVD en GNU/Linux desde la línea de comandos

26 febrero, 2017 En # CODE

Si dispones de algunas películas en DVD puedes montarlo en un contenedor mkv (vídeo, audios, subtitulos) de forma sencilla y sin necesidad de herramientas gráficas. Pero antes tendrás que extraer el contenido de los DVDs originales.

Instalando algo de software:

 

apt-get install  vobcopy libdvdread4 ffmpeg dvdrip-utils libdvd-pkg ogmrip

 

Si andas en ubuntu, xubuntu, … tendrás que instalar algo así como esto:

apt-get install xubuntu-restricted-extras

Luego:

vobcopy -m

 

Si se instala dvd css no saldrá el mensaje que me ha salido a mi antes de instalar el tinglado (se ve en la primera captura).

El programa OGMrip tambien podrá ser de utilidad. Aunque no es para línea de comandos.
