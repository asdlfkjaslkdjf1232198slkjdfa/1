MQTT-CLI para conectar a un broker publico

26 febrero, 2017 En # CODE

mqtt-cli es un cliente MQTT sencillo y para línea de comandos. Está escrito en nodejs. Para instalar el cliente podemos hacerlo así:

 


npm install mqtt-cli -g

 

Lo podemos poner a escuchar un canal de un broker público (por ejemplo broker.hivemq.com:1883) así:



mqtt-cli broker.hivemq.com:1883 series "" -w

 

En otro terminal podemos enviar mensajes al canal, por ejemplo este:
