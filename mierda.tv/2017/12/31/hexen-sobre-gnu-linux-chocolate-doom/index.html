Hexen – Sobre GNU/Linux – Chocolate-doom

31 diciembre, 2017 En # GAMES

En 1995 salió Hexen usando el motor de Doom de ID software. En la actualidad en GNU/Linux puede jugarse a Hexen ya que existen engines libres como chocolate-doom que si le indicamos el archivo WAD del juego original pues lo utilizarán.

En Debian es posible instalar chocolate-doom así:

# apt-get install chocolate-doom

Descargar el archivo WAD de Hexen así:

$ wget https://mierda.tv/descargas/HEXEN.WAD

Ejecutar chocolate para disfrutar de Hexen:

$ chocolate-hexen -iwad HEXEN.WAD

Disfruta :).
