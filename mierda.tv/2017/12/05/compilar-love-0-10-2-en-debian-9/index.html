Compilar Love 0.10.2 en Debian 9

5 diciembre, 2017 En # CODE

Debian 9 por defecto en los repositorios trae love. Es posible instalar love simplemente haciendo “apt install love” pero … tendremos una versión de love algo más vieja y por tanto sin por ejemplo soporte para videos ogg theora.
Love nos permite poder programar de forma sencilla juegos en 2D y según dicen las malas lenguas mola mucho.
 
Lo cierto es que tiene una apariencia muy buena y dan ganas de utilizarlo y empezar a programar utilidades con Love.
 
No obstante para ello es mejor compilarse la última versión. Por eso el motivo de dejar los pasos relatados en esta entrada para que en futuras entradas sobre love se pueda enlazar a como se monta esto.
 

# apt-get install build-essential automake libtool libphysfs-dev 
# apt-get install libsdl1.2-dev libopenal-dev liblua5.1-0-dev libdevil-dev libmodplug-dev
# apt-get install libfreetype6-dev libmpg123-dev libvorbis-dev libmng-dev libxpm-dev 
# apt-get install libxcursor-dev libxxf86vm-dev libtheora-dev libluajit-5.1-dev 
# apt-get install liblua5.2-dev liblua5.3-dev lua5.1 lua5.2 lua5.3 libsdl-gfx1.2-dev libsdl2-dev

$ wget "https://bitbucket.org/rude/love/downloads/love-0.10.2-linux-src.tar.gz"
$ tar xfvz love-0.10.2-linux-src.tar.gz 
$ cd love-0.10.2/
$ ./configure --with-lua=lua
$ make
# make install
