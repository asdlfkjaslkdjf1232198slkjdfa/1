Compilar Scummvm 2.0.0 en GNU/Linux

29 diciembre, 2017 En # CODE, # GAMES

Para disfrutar de bastantes juegos en GNU/Linux podemos usar Scummvm. Es interesante hacerlo con la versión 2.0.0 ahora mismo en vez de las anteriores que lo mismo encontramos si en vez de compilar tiramos de repositorio en sistemas GNU/Linux como por ejemplo Debian.

Algunas dependencias no necesariamente obligatorias pero interesantes:

# apt-get install libfaad-dev libmpeg2-4-dev libvorbisidec-dev libfluidsynth-dev libsdl-net1.2-dev 
# apt-get install g++ make libsdl2-dev libjpeg62-turbo-dev libogg-dev libvorbis-dev libflac-dev
# apt-get install libmad0-dev libpng-dev libtheora-dev libfreetype6-dev zlib1g-dev wget

Para compilar Scummvm 2.0.0 en Debian 9 o en cualquier sistema GNU/Linux:

$ wget "http://www.scummvm.org/frs/scummvm/2.0.0/scummvm-2.0.0.tar.gz"
$ tar xfvz scummvm-2.0.0.tar.gz
$ cd scummvm-2.0.0/
$ ./configure --enable-all-engines
$ make clean
$ make
# make install
