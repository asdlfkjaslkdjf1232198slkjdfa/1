10 “truquillos” para usuarios/as de GNU/Linux – Vol 7.

3 diciembre, 2017 En # SYSTEM

1. Sin limites de descarga en Mldonkey server

El siguiente comando nos ayudará:

set max_hard_download_rate 0

 

2. Ver de forma molona los tamaños de archivos, directorios

En vez de usar du -h podemos usar ncdu y disfrutar de algo más vistoso.
Se puede instalar en debian con:

apt-get install ncdu

Otras similares son htop en vez de top y dfc en vez de df.
 

3. Mantener vivas las sessiones ssh para evitar el timeout desde el lado cliente

Si no existe creamos en ~/.ssh/ un archivo llamado config y en su interior:

ServerAliveInterval 120

 

4. Compilar xmppipe,  un bot xmpp para notificar en salas o a cuentas

git clone https://github.com/strophe/libstrophe
cd libstrophe
bash bootstrap.sh
./configure
make
make install
cd ..
git clone https://github.com/msantos/xmppipe
cd xmppipe
make
make install
cp -pRv xmppipe /usr/local/bin

 

5. Cambiar fecha y hora de un archivo

 
Ejemplo:
 

touch -t 201712022045 /var/www/html/peliculas/Their_finest-2016-Spanish_English_Subs-Fanta.mkv

 

6. Varias formas de cambiar el propietario de un archivo/directorio

 
Si nos confundimos poniendo . en vez de : sirve igual:

chown usuario:usuario archivoejemplo.zip
chown usuario.usuario archivoejemplo.zip

 

7. Comprobar configuración de Nginx

 

nginx -t

 

8. Capturar peticiones DNS

 

tcpdump -nt -i eth0 udp port 53 > captura.cap

cat captura.cap | grep "xxx.xxx.xxx.xxx.53:" | rev | cut -d "(" -f 2 | cut -d " " -f 2 | cut -d "." -f 2-100 | rev

Cambia las xxx por tu IP.
 
 

9. Cambiar el servidor DNS si usamos resolvconf

 
Lo normal viene siendo editar el archivo /etc/resolv.conf y añadir allí algo como esto:

nameserver 178.32.17.208

Si se está usando resolconf como demonio este sobreescribirá lo que se añada y dejará de valer. Para ello lo que se hace es meterlo en el archivo base que en muchas distros estará en /etc/resolvconf/resolv.conf.d/base
Tras ello se ha de reiniciar resolvconf así por ejemplo: /etc/init.d/resolvconf restart
