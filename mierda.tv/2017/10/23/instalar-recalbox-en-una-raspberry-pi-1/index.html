Instalar Recalbox en una Raspberry Pi 1

23 octubre, 2017 En # GAMES

Desde aquí descargamos la imagen (img.xz). La que corresponda con la raspberry pi 1 (o la que tengamos).

Para descomprimirla tendremos que tener instalado xz (en debian se puede instalar así: apt-get install xz-utils).

Nos bajamos de https://etcher.io/ el programar etcher para grabarla en la tarjeta SD.

Recomiendan una de 16Gb. Yo tengo ahora mismo a mano solamente una de 8Gb.

Una vez grabada la tarjeta SD (para raspberry pi 1 eran SD, en la raspberry pi 3 es una micro SD por ejemplo) la ponemos en la raspberry PI y la encendemos.

En mi caso he usado 2 game pads: Thrustmaster Dual Analog 4 (por unos 16 € los puedes encontrar).
He tenido que configurar los pads (los detecta bien) y también el sonido.
También en opciones avanzadas se puede activar el overclocking extremo para que se mueva un poco mejor (es una raspberry pi 1).

Finalmente los juegos pueden meterse en la unidad que se monta llamada SHARE cuando metemos la SD en el lector de tarjetas del PC. Allí podemos meter las roms de los juegos.

Un bueno juego para probar es el cadilacs and dinosaurs https://www.emuparadise.me/M.A.M.E._-_Multiple_Arcade_Machine_Emulator_ROMs/Cadillacs_and_Dinosaurs_(World_930201)/12472 que sin el overclocking no chutará fino pero si se hace overclocking chutará fino filipino.

Puede verse un vídeo aquí: https://mierda.tv/descargas/recalbox.mpeg

No olvides una vez metes nuevos juegos darle al menú scraper. Eso descargará información sobre el juego si lo tienes contectado a internete así como un screenshot.
