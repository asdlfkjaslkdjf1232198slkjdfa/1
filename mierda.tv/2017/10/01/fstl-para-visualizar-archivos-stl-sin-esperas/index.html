fstl – Para visualizar archivos STL sin esperas

1 octubre, 2017 En # 3D

El programa fstl es una maravilla. En vez de abrir un programa que nos permita editar un archivo STL es mucho más rapido simplemente usar un visor para esas tareas. Cuando simplemente deseas visualizar un archivo STL fstl es tu programa.
 
En Debian GNU/Linux 9 puede instalarse así:
 

# apt-get install fstl

 
No obstante yo con el programa instalado desde el repositorio de Debian he tenido problemas y me casca. Si a alguien más le ocurre lo mismo se ha de reportar.
 
La forma pues si te casca es compilarlo. Merece la pena ya que ganarás mucho tiempo a la larga si estás trabajando con este tipo de archivos.

 

# apt-get install qmake qt4-qmake cmake qt5-default
$ git clone https://github.com/mkeeter/fstl
$ cd fstl
$ mkdir build
$ cd build
$ qmake ../qt/fstl.pro
$ cmake ..
$ make -j8
$ ./fstl
# cp -pRv fstl /usr/bin/
