10 Juegos libres molones para GNU/Linux

25 noviembre, 2017 En # GAMES

1. Unknown Horizons

Un juego de estrategia libre rollo colonization. Has de meterle unas cuantas horas para ir viendo como se juega.
Es posible instalar el juego si usas debian 9 así: apt-get install unknown-horizons

Website: http://unknown-horizons.org/

2. Widelands

Un juego de estrategia rollo recursos. Libre y con parte multiplayer en LAN/Internet. Basado en Settlers II es un juego entretenido pero con una curva de aprendizaje de un tiempo. Ideal para quienes les gusten estos juegos de estrategia y de cazadores/recolectores.
En Debian 9 puede instalarse así: apt-get install widelands

Website: https://wl.widelands.org/

3. 0 A.D.

Un nombre raro pero un pedazo de juego. Puedes jugar online o contra la maquina. Juego de estrategia que ya tiene unos años pero que sigue mejorando con el tiempo. En Julio de este año se sacó una nueva versión.
Puedes instalar en Debian el juego o puedes bajar la última versión de la web. Lo ideal es siempre usar la misma para jugar con el resto de gente.

Website: https://play0ad.com/

4. Seven Kingdoms – 7kaa

En Debian se puede instalar así: apt-get install 7kaa . Un juego de estrategia que requiere inicialmente de un poco de paciencia para hacerte con los controles pero que luego mola mucho.

Website: https://github.com/the3dfxdude/7kaa

5. Zero-K

Este tiene ya unos años y seguro que lo has jugado alguna vez. Nunca es tarde para jugarlo de nuevo.

Website: http://zero-k.info/

6. Billiards


Website: http://www.nongnu.org/billiards/
En debian puede instalarse así: apt-get install billard-gl
Merece la pena invertir algo de tiempo para saber como funciona.

7. FreeDink


Website: http://www.gnu.org/software/freedink/
En debian puede instalarse así: apt-get install freedink
Puedes matar los cerdos.

8. Briquolo


Website: http://briquolo.free.fr/en/index.html
En debian puede instalarse así: apt-get install briquolo
Es muy divertido hasta que coges algo que te hace verlo en primera persona y por tanto muy difícil controlar la bola.
Muy creativo y si le metes algo de tiempo divertido.

9. Eat The Whistle


Website: http://www.ggsoft.org/etw/
En debian puede instalarse así: apt-get install etw
Divertido y retro. Para quienes gustan de juegos de deportes de vista aérea. Requiere revisar los controles para saber como jugar. No pasa nada por eso, toda la vida para jugar a un juego tenias que ver con que teclas jugar. Forma parte del juego eso en vez de tanto juego sabiendo ya las teclas por apretar ya que te dan un mando.

10. Tu juego

Un juego genial que usará mucha gente. disfrutará y disfrutarás si lo compartes.
