mtm – Terminal multiplexor sencillo

14 mayo, 2017 En # CODE

Este programa es crema de la buena. Sencillo, compatible, estable y de apenas 1000 líneas de código.
  
Este programa nos permite tener varias terminales dividiendo la pantalla en horizontal/vertical. Es sencillo ya que no se necesita recordar miles de comandos, solamente 3: split, close y changefocus.
  
Lo que viene a ser lo más usual y necesario. Poder dividir la terminal para sacar nuevas (en horizontal o vertical), cerrar alguna que hemos partido y poder cambiar el foco sin usar el ratón.
  
Se puede compìlar e instalar así:
  

$ git clone https://github.com/deadpixi/mtm
$ cd mtm
$ make
# make install

  
Lo bonito es que es muy sencillo manejarlo. Como habitualmente con control+D vamos a poder cerrar la terminal o ir saliendo de las sesiones que tengamos de modo que cerrar una terminal es igual.
  
Partir en horizontal es con [control+g]+h o en vertical [control+g]+v.
  
Y para cambiar el foco de una ventana a otra: [control+g]+left/right/down/up
  

Una maravilla de programa. Sencillo y con lo necesario funcionando guay.
