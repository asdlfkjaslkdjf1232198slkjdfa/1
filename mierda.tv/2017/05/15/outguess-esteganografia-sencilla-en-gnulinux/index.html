Outguess – Esteganografía sencilla en GNU/Linux

15 mayo, 2017 En # CODE

Con el software outguess se pueden camuflar cifrados archivos en un jpg por ejemplo, es decir, meter por ejemplo otro jpg dentro del archivo de imagen y que solamente se visualice el que usamos como contenedor (y que al mismo tiempo el que añadimos esté cifrado).
  
Instalar este software en debian se puede desde los repositorios así:
  

# apt-get update
# apt-get install outguess

  
Veamos un ejemplo partiendo de una foto limpia y una foto que será nuestro mensaje
  

Foto limpia:
  

  
Mensaje que queremos incluir en su interior:
  

  
El comando para meter una dentro de la otra será este:
  

outguess -k "12345" -d verdadero.jpg foto-limpia.jpg foto-sucia.jpg

  
La clave es 12345 y nos creará una nueva foto llamada foto-sucia.jpg que es la que contiene la otra imagen cifrada con la clave 12345 en su interior.
  
Está es la foto:
  

  
Si nos la bajásemos podríamos descifrar el mensaje simplemente así:
  

outguess -k "12345" -r foto-sucia.jpg exit.jpg

  
Tendríamos una nueva imagen llamada exit.jpg.
  
Un poco más de lío. Meter 2 textos, uno verdadero y otro falso y ambos cifrados con claves diferentes:
  
Creamos los 2 textos así por ejemplo:

echo "hola mundo este mensaje es un mensaje falso." > falso.txt
echo "este es un mensaje con el contenido verdadero :)" > verdadero.txt

  
Ahora los metemos en la imagen foto-limpia.jpg con la clave “falso” para el texto falso.txt y clave “verdadero” para verdadero.txt saliendo la imagen resultante como foto-sucia.jpg que contendrá ambos mensajes.
  

outguess -k "falso" -d falso.txt -E -K "verdadero" -D verdadero.txt foto-limpia.jpg foto-sucia.jpg

  
Para extraer el mensaje falso:
  

outguess -k "falso" -r foto-sucia.jpg falso.txt

  
Para extraer el mensaje verdadero:
  

outguess -k "verdadero" -e -r foto-sucia.jpg verdadero.txt

  
Saludos cordiales.
