LÖVE – Un maravilloso framework para hacer juegos en 2D

17 mayo, 2017 En # CODE

Este framework es bueno para aprender LUA o iniciarse con este lenguaje. Aparte de ser mejor persona LÖVE es un framework potente y listo para gente que quiera desarrollar un juego en 2D.
  
Actualmente usa estas librerias:
  
Lua – Scripting.
OpenGL – Hardware accelerated graphics.
DevIL – Image loading.
SDL – OS abstraction, mouse input, keyboard input.
LuaSocket – Network.
FreeType 2 – Font loading and rasterization.
PhysicsFS – Filesystem management + support for archive files.
OpenAL Soft – Audio playback.
mpg123 – MP3 support.
libogg/vorbis – OGG support.
libmodplug – Tracker formats support.
Box2D – Physics
  
Algunos juegos realizados con LÖVE son realmente y visualmente agradables. Por ejemplo Mari0 al estilo Portal:
  

  
También proyectos tan locos y disparatados como http://www.ohmygiraffe.com/ o Warlock’s Tower.
  
La web del proyecto: https://love2d.org/
  
Es un Framework sencillito y se comienza a trabajar rapidamente.
  
Mostrar un texto es así de simple:

    function love.draw()
    love.graphics.print("Hello World!", 400, 300)
    end

  
Mostrar una imagen:
  

    function love.load()
    whale = love.graphics.newImage("whale.png")
    end
    function love.draw()
    love.graphics.draw(whale, 300, 200)
    end

  
Reproducir un sonido:

    function love.load()
    sound = love.audio.newSource("music.ogg")
    love.audio.play(sound)
    end

  

Una herramienta para todo aquel que se asuste trabajando con C y SDL y que prefiera algo más sencillo para perder menos tiempo.
