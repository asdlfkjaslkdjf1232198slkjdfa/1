Typecatcher – Cientos de tipografías a golpe de clic

17 mayo, 2017 En # CODE

Typecatcher es un sencillo pero efectivo programa que nos sirve para de golpe y porrazo poder añadir nuevas tipografías en nuestro sistema.
  
Las tipografías de https://fonts.google.com/ son libres y es buena cosa poder navegar rápidamente entre ellas y descargar la que necesitemos cuando queramos.
  
Tambien es interesante conocer la web Dafont en la que encontraremos muchas otras y que se presta a que alguien desarrolle un typecatcher que se nutra de allí o algo así.
  
En Debian/Ubuntu/… se puede instalar desde los repositorios así:
  

# apt-get install typecatcher
