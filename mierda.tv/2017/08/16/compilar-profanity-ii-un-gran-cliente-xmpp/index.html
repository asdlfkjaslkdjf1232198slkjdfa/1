Compilar Profanity II – Un gran cliente XMPP

16 agosto, 2017 En # CODE

Hace un año en esta entrada se hablaba de como instalar profanity en debian 8. Este año toca comentar como se puede hacer en debian 9.
  
La forma sencilla es: apt-get install profanity . La otra forma es compilar la última versión.
  

  
Instalar dependencias
  

apt-get install autoconf-archive libreadline-dev libstrophe-dev libstrophe0 git dh-autoreconf libssl-dev libssl-dev libexpat1-dev libncursesw5-dev libglib2.0-dev libnotify-dev libcurl3-dev libxss-dev libotr2-dev

  
Compilar openssl
  

git clone https://github.com/openssl/openssl
cd openssl
./config
make
make install

  
Descargar y compilar profanity
  

wget "http://www.profanity.im/profanity-0.5.1.tar.gz"
tar xfvz profanity-0.5.1.tar.gz 
cd profanity-0.5.1
git clone https://github.com/strophe/libstrophe
cd libstrophe
./bootstrap.sh 
./configure
make
make install
cd ..
