Robots de papel

5 noviembre, 2016 En # SHIT

En el hackerspace “Laboratorio de creación digital” de Oporto (Portugal) han liberado varios modelos imprimibles de robots que puedes imprimir, recortar, pegar y tener en algún lado adornando.

Han colgado los modelos en svg y pdf en un repositorio en github. Se puede clonar el repo así:

git clone https://github.com/lcdporto/electronic-paper-toys

Para poder editar los svg es bueno disponer de por ejemplo Inkscape.

apt-get install inkscape

Este tipo de iniciativas están geniales para poder replicarlas en otros centros. Es bueno que se compartan para que se puedan utilizar y editar.

Es un buen material para trabajar con chavales. Por un lado aprenden a usar Inkscape al mismo tiempo que luego trabajan con las manos para recortar, pegar, …
