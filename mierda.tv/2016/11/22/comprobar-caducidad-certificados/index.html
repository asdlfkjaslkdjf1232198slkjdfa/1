Comprobar caducidad certificados

22 noviembre, 2016 En # SYSTEM

Un sencillo script para comprobar la fecha de caducidad de determinados certificados https pasando una url.

El script lo que hace es tirar de curl. Esto sirve para pasárselo a un listado de urls que tengamos y del tirón saber la fecha en la que caducarán los certificados.

#!/bin/bash
# Obtenemos la fecha de caducidad del certificado de una url
url=$1 # ejemplo https https://lavida.sitioweb.com/
fc=$(curl -vvI $url 2>&1 | grep "expire date:" | cut -d ":" -f 2 | cut -d " " -f 3-5)
fc=$(date -d "$fc" +%Y/%m/%d)
echo "$url;$fc"
