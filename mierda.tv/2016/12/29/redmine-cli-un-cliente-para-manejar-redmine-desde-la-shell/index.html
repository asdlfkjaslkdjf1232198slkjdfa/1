Redmine cli – Un cliente para manejar redmine desde la shell

29 diciembre, 2016 En # CODE

Información sobre la Rest api: http://www.redmine.org/projects/redmine/wiki/Rest_api

Si tienes acceso a un redmine haz login y encontrarás en /my/account la key api y si pinchas allí sobre tu identificador o miras la url verás tu ID de usuario (un número). Esto lo vamos a necesitar para crear un fichero de configuración más adelante.

Es importante saber que con la versión 2.6 de python esto va bien. Instalaremos redmine-cli así:

# apt-get install python-pip
# pip-2.6 install redmine-cli

En el .bashrc del usuario que queramos usar con redmine-cli podemos crear un alias como este:

alias python=/usr/bin/python2.6

 

Si todo va bien vamos a crear un archivo llamado .redmine-cli en nuestro home con un contenido como este:

 

[default]
 key=5237223d239072d648c554475c974b0a832791aa12
 my_id=149
 root_url=https://laurl.dondeestararedmine.com

 

Ahora ya simplemente a disfrutar:

redmine issues
