Instalar Docker sobre debian 8 jessie

8 diciembre, 2016 En # SYSTEM

Tras la charla del señor miyagi en la brecha digital me entró el gusanillo de probar docker. Es por eso que he creado esta entrada sobre como instalarlo en debian 8 para posteriormente en otras entradas no tener que repetir el “como instalar docker” y simplemente enlazar este post.

Comprobamos la versión del kernel así:

uname -r

Tendría que ser superior a la 3.10

Si comprobamos con -a veremos que la versión de debian tiene que ser de 64.

Comprobamos que estamos en debian 8 por ejemplo así:

 

cat /etc/issue

 

Añadimos los repositorios añadiendo repo de backports en /etc/apt/sources.list

 

deb http://http.debian.net/debian wheezy-backports main

 

Luego actualizamos así:

 

apt-get update

 

Instalamos software que vamos a necesitar:

 

apt-get install apt-transport-https ca-certificates gnupg2

 

Añadimos en /etc/apt/sources.list el repo de dockers:

 

deb https://apt.dockerproject.org/repo debian-jessie main

 

Guardamos los cambios en el archivo y añadimos la clave GPG así:

 

apt-key adv --keyserver hkp://p80.pool.sks-keyservers.net:80 --recv-keys 58118E89F3A912897C070ADBF76221572C52609D

 

Ahora actualizamos e instalamos docker:

 

apt-get update
apt-cache policy docker-engine
apt-get install docker-engine

 

Para que funcione tendremos que añadir a grub lo siguiente en /etc/default/grub

 

GRUB_CMDLINE_LINUX_DEFAULT="cgroup_enable=memory swapaccount=1"

 

Tras guardar ejecutamos:

 

update-grub

 

Reiniciamos el sistema y ya está listo para usarse si todo va bien.

 

Si queremos añadir un usuario para usar docker en vez de usar root lo podemos hacer así (el ejemplo es el usuario fanta):

 

groupadd docker
gpasswd -a fanta docker
/etc/init.d/docker restart

 

Información extraída de la charla del señor Miyagi en el centro social la brecha y de https://docs.docker.com/engine/installation/linux/debian/
