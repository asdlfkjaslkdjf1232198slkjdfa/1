Convertir a png una presentación creada con libreoffice impress

23 septiembre, 2016 En # CODE

Exportar todas las slides de una presentación a pdf es muy sencillo y se puede hacer tanto en línea de comandos como mediante la GUI. La cosa es que algunas veces por algún motivo vamos a necesitar exportar posiblemente alguna slide y allí es donde podemos tranquilamente utilizar libreoffice y convert.

 

Se necesita tener instalado libreoffice e imagemagick. El proceso puede ser similar a este:

libreoffice5.2 --headless --convert-to pdf 7_vidas_extra_tiene_un_juanker.odp 
convert -density 150 7_vidas_extra_tiene_un_juanker.pdf -quality 90 output.png


La salida será: output-0.png output-1.png output-2.png …
