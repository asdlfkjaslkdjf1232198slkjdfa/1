Abrir y editar imágenes IFF en GNU/Linux

1 septiembre, 2016 En # CODE

IFF–ILBM es un formato de imagen que fue popular en los tiempos de oro del ordenador amiga. Este tipo de archivos son complicados de abrir con los editores tradicionales por lo que vamos a tener que instalar en GNU/Linux el programa imview para poder visualizarlos.

 

Imview en debian podemos instalarlo así:

 

# apt-get install imview

 

 

Para editar directamente este formato podemos hacerlo con un plug-in para GIMP. El plug-in para gimp IFF-ILBM nos permitirá usar el editor de mapas de bits Gimp para abrir este tipo de imágenes y poder trabajar con ellas.

 

 

Si instalamos las dependencias necesarias complicarlo será sencillo. EL problema es que al realizar el make install esto colocará en /root/.gimp/plugins/ el plugin y eso quiere decir que si queremos probarlo tenemos que ejecutar Gimp como root. Otra opción más segura y recomendad es mover el contenido al directorio oculto de gimp en tu home.

Estás 2 herramientas nos facilitarán la vida a la hora de trabajar con imágenes de esos años.
