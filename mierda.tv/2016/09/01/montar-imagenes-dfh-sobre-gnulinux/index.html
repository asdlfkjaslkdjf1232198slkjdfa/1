Montar imágenes DFH y ADF sobre GNU/Linux.

1 septiembre, 2016 En # CODE

Montar una imagen de disco en formato FFS (Fast File System). Los discos duros para los ordenadores Amiga solían formatearse en este sistema de ficheros. En GNU/Linux es posible montar tanto los discos duros como imágenes de estos.

Para probarlo podemos bajarnos algún disco desde aquí http://classicwb.abime.net/classicweb/download.htm así:

 

$ wget "http://download.abime.net/classicwb/ClassicWB_LITE_v28.zip" -O shit.zip
$ unzip shit.zip
$ cd ClassicWB_LITE_v28

 

Para montarlo así:

 

$ mkdir /home/usuario/disco
# mount -t affs -o loop System.hdf /home/usuario/disco/

 

No tiene más misterio pero es bonito que pueda hacerse de forma tan sencilla.

Se puede usar también para imágenes adf (las de los disquetes). Ejemplo:

 

mount -t affs -o loop mm1.adf disco1/
