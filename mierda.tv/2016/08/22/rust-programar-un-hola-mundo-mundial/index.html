Rust – Programar un hola mundo mundial

22 agosto, 2016 En # CODE

Para poder programar en Rust con un simple editor de textos nos valdrá. Para luego compilar si que vamos a necesitar el compilador rustc.

En debian podemos montar todo el tinglado así:

 

# apt-get install sudo # adduser nombreusuario sudo

Añadimos al archivo /etc/sudoers el nombre de nuestro usuario con por ejemplo nano (también puedes con vi, joe, emacs o lo que te de la real gana).

 

# nano /etc/sudoers

Esta línea de ejemplo:

nombreusuario ALL=(ALL) ALL

Comenzamos con la instalación de rust en el sistema:

 

$ curl -sSf https://static.rust-lang.org/rustup.sh | sh

Veremos algo así como ‘Rust is ready to roll’ en el caso de que esté ok.

Para ver la versión:

 

rustc --version

Ejemplo de salida:

 

rustc 1.10.0 (cfcb716cf 2016-07-03)

Canal de IRC en el que conseguir ayuda para principiantes/as: irc://irc.mozilla.org/#rust-beginners

 

Crear un hola mundo:

 

mkdir proyectos
cd proyectos/
mkdir hola_mundo
cd hola_mundo/
nano hola.rs

 

En el interior:

fn main() {
    println!("Hola mundo");
}

 

Lo compilamos así:

 

rustc hola.rs

 

Lo ejecutamos por primera vez así:

 

./hola

 

Ya está. Veremos el hola mundo en pantalla. Si utilizamos nano quizás nos venga bien esto: https://github.com/rust-lang/nano-config y esto para saber como meterlo: https://mierda.tv/2016/08/14/nano-un-editor-de-textos-para-editar-textos/
