Instalamos GNUplot así:

apt-get install gnuplot

Un ejemplo de uso que creará un jpg se puede ver aquí:

$ echo 'set term jpeg; set title "Temperatura"; set ylabel "Grados"; set xlabel "Últimas 48h";set grid;plot "datos48horas.txt" title "" with lines'| gnuplot > datos48horas.jpg

Obtiene los datos del archivo de texto. En este ejemplo esos datos son cada línea una hora.

Esto solamente es un ejemplo de como utilizar gnuplot. Lo cierto es que tiene mucha más manteca el asunto pero puede venir bien para iniciarse.
