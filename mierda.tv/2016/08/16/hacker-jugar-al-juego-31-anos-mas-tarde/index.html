Hacker – Jugar al juego 31 años más tarde

16 agosto, 2016 En # GAMES

Este juego se realizó 2 años después de que se estrenase la película “WarGames” (juegos de guerra).  Para poder disfrutarlo en GNU/Linux vamos a tener que emular un ordenador Spectrum (aunque existe versión para otros ordenadores). Los pasos son los siguientes.

Instalar el emulador fuse en debian

Esto instalará el emulador y algunas roms:

# apt-get install fuse-emulator-common fuse-emulator-utils spectrum-roms

 

La lista de ROMs incluidas es esta:

48.rom – Original 48K ROM
128-0.rom – 128K ROM 0 (128K editor and menu)
128-1.rom – 128K ROM 1 (48K BASIC)
128-spanish-0.rom – Spanish 128K ROM 0 (128K editor & menu)
128-spanish-1.rom – Spanish 128K ROM 1 (48K BASIC)
if1-2.rom – Interface 1 v2 ROM
plus2-0.rom – +2 ROM 0 (128K editor and menu)
plus2-1.rom – +2 ROM 1 (48K BASIC)
plus3-40-0.rom – +2A/+3 v4.0 ROM 0 (128K editor)
plus3-40-1.rom – +2A/+3 v4.0 ROM 1 (128K syntax checker)
plus3-40-2.rom – +2A/+3 v4.0 ROM 2 (+3DOS)
plus3-40-3.rom – +2A/+3 v4.0 ROM 3 (48K BASIC)
tc2048.rom – Original 48K ROM modified for the Timex TC2048

Las ROMs se encontrarán en /usr/share/spectrum-roms/

Montar el tinglado

 

Creamos un directorio en nuestro home llamado por ejemplo fuse y dentro otros 2 llamados roms y tapes:

 

mkdir fuse;cd fuse;mkdir roms tapes

 

Copiamos las roms al directorio roms así:

 

cp -pRv /usr/share/spectrum-roms/* roms/

 

Nos bajamos el juego “Hacker” así para se quede en el directorio tapes:

wget -q "https://mierda.tv/wp-content/uploads/2016/08/Hacker-1985-Activision.tap_.tar.gz" -O tapes/Hacker-1985-Activision.tap.tar.gz

 

Lo descomprimimos:

 

cd tapes
tar xfvz Hacker-1985-Activision.tap.tar.gz
rm Hacker-1985-Activision.tap.tar.gz -rf

 

Vamos a jugar, tus problemas déjalos

Ejecutamos fuse:

 

fuse

 

Nos vamos a: options –> select ROMs –> Spectrum 48K …

Seleccionamos la rom del directorio roms que hemos creado y en el que las hemos copiado. En este caso 48.rom

Ahora vamos a cargar la cinta. Nos vamos a Media –> Tape –> Open …

Seleccionamos la cinta del directorio tapes que hemos creado antes.

Presionamos [F5] para resetear. Luego presionamos la tecla [j] y seguidamente [control]+[p] y [control]+[p] [enter]

Ya ha comenzado el juego :).
