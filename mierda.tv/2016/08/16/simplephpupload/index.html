simple php upload – Compartir rápidamente

16 agosto, 2016 En # CODE

Simple PHP upload es un script escrito en php que te permitirá subir archivos de forma sencilla y compartirlos.
Esto funciona como el típico sitio donde subes cosas y te da una url para que puedas compartir solamente que en vez de subirlo a un servidor que no es tuyo lo subes a tus maquinas.

 

Este script puede bajarse desde aquí: https://github.com/muchweb/simple-php-upload

 

Es bueno configurarlo para que guarde los archivos en un directorio especifico y al mismo tiempo configurar nginx (o el servidor web que usemos) para que no permita que cualquier desgraciado de la vida pueda subir cosas.
Aquí se pueden ver configuraciones de nginx de ejemplo: https://github.com/muchweb/simple-php-upload#server-configuration

 

Aquí puedes ver una captura del tinglado en acción:

 

 

No tiene mucho misterio y viene bien para compartir algo rápidamente.

 

Es posible indicar que los archivos se borren cada x tiempo o se puede desactivar eso y meter una regla de cron que permita borrar los archivos más viejos de x tiempo cada día.

No somos nadie pero este script es buena cosa.
