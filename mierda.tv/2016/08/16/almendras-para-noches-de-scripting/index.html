Almendras para noches de scripting

16 agosto, 2016 En # SHIT

Las almendras son buena cosa para noches de scripting en las que necesitas hidratarte bien y al mismo tiempo estar metiendo algo a la boca cada x tiempo. En vez de pipas puedes optar por almendras.

 

Dependencias

  • Almendras
  • Sal
  • Aceite de oliva virgen extra
apt-get install almedras sal aceite

Pasos

  • Apaleamos con amor el árbol para que caigan los alemendrucos.
  • Partimos los almendrucos con un martillo.
  • Cuando tenemos muchas almendras vamos a proceder a pelarlas.
  • Una vez peladas las freiremos con un poquito de aceite.
  • Finalmente le añadimos un poco de sal.
./configure; make; make install

Hack & Tricks

- Girando el almendruco se partirá mejor.
- Las almendrás las ponemos en agua hirviendo para poder quitarles la "cascarilla interior".
- La sal al final.
- Guardar en un recipiente de cristal cerrado herméticamente.
