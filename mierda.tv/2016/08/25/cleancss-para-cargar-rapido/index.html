Cleancss para cargar rápido

25 agosto, 2016 En # SYSTEM

Aparte de “las magias” que se pueden hacer con Nginx para cachear contenidos siempre estará bien que las hojas de estilo las mandemos comprimidas y en una línea. No son buena cosa para trabajarlas a nivel de desarrollo pero si para ponerlas en producción y que el contenido tarde menos en cargar.

Los buscadores penalizan webs que cargan lento. Para que esto no pase cleancss permite ganar ese algo más de tiempo para que la experiencia de usuario no sea una mierda y se piren esperando a que el contenido este disponible ante sus ojos.

Instalación en debian:

apt-get install cleancss

Uso sencillo:

cleancss -o nuevoconmenorpeso.css archivooriginal.css

 

El archivo resultante será nuevoconmenorpeso.css y tendría que ocupar mucho menos.
