fspy – Monitorizar un archivo/directorio determinado

23 agosto, 2016 En # CODE

fspy es un software escrito por Richard Sammet (e-axe) que se quedó en la versión 0.1.1 y que lleva sin actualizarse desde 2010. Su autor tenia un blog hace años pero lo abandonó. Por suerte aún se puede recuperar del archivo de waybackmachine aquí: http://web.archive.org/web/20100712090331/http://mytty.org/fspy/

 

Este software nos permite monitorizar un archivo especifico para ver cuando se lee, escribe, mueve, … también un directorio entero.

Instalar en debian

apt-get install fspy

 

En node existe también un fspy: https://www.npmjs.com/package/fspy

 

Motivos para usar esto

 

Supongo que cuando tienes archivos importantes y quieres controlar su acceso. Por ejemplo la base de datos de keepassx.

De esta forma si alguien ha entrado a tu maquina podrás ver si ha leído, escrito, … algún archivo.
