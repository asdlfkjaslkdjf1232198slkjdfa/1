Fiche – Tu “pastebin” desde línea de comandos

12 agosto, 2016 En # CODE

Fiche permite que puedas enviar textos desde la consola y que estos sean visibles en una url.
Esto es crema para quienes andamos todo el día en la shell y deseamos en un momento dado compartir por ejemplo un listado o el contenido de un archivo de texto plano.

Lo justo y necesario

 

  • Instalar git
  • Instalar netcat.
  • el compilador gcc
  • Un servidor web como por ejemplo apache2 o nginx (no voy a entrar en juicios de cual es mejor).
  • fiche
  • Instalar screen

 

Descargar fiche

 

Antes de nada las dependencias cumplirlas. En debian así:

 

# apt-get install screen git gcc netcat nginx make

 

Es posible descargar fiche desde aquí: https://github.com/solusipse/fiche . Puedes clonarlo en un directorio de tu servidor así:

 

$ git clone https://github.com/solusipse/fiche

 

Compilar fiche

 

$ make
# make install

 

Algunos Usos desde la terminal cliente

 

$ ls -la | nc localhost 9999
$ cat file.txt | nc solusipse.net 9999
$ echo just testing! | nc code.solusipse.net 9999

 

Hemos de tener en cuenta el puerto. Ha de ser el que indiquemos.

Hemos de ejecutar fiche por ejemplo en una screen para que nos permita conectarnos.

No tiene mucho misterio el tema y es buena cosa. De esta forma conseguiremos disponer de nuestro propio pastebin desde consola.
