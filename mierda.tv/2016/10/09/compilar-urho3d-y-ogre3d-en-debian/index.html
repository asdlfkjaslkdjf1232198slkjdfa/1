Compilar Urho3D y Ogre3D en Debian

9 octubre, 2016 En # GAMES

Estos 2 motores son libres. El primero está basado en el segundo y es más sencillo de montar el tinglado. Sirven para 3D y para 2D.

 

Si deseamos compilar Urho3d podemos hacerlo así:

Instalar dependencias, descargar Urho3d y compilar Urho3d.

# apt-get install libx11-dev libxrandr-dev libasound2-dev git cmake make
$ wget https://github.com/urho3d/Urho3D/archive/1.6.tar.gz
$ tar xfvz 1.6.tar.gz
$ cd Urho3D-1.6/
$ cmake .
$ make
# make install

 

Nos instalará unos 40 y tantos ejemplos. Podemos ejecutarlos y verlos escribiendo 01 pulsar tabulador, 02, pulsar tabulador, 03, … todos los ejemplos empiezan con un número.

 

Si deseamos compilar Ogre3D podemos hacerlo así:

Instalar dependencias, descargar Ogre y compilar Ogre.

# apt-get install libfreetype6-dev libboost-date-time-dev libboost-thread-dev nvidia-cg-toolkit libfreeimage-dev zlib1g-dev libzzip-dev libois-dev libcppunit-dev doxygen libxt-dev libxaw7-dev libxxf86vm-dev libxrandr-dev libglu-dev cmake-gui cmake libsdl1.2-dev libdevil-dev libfreetype6-dev libgtkmm-2.4-dev libglademm-2.4-dev mercurial
$ hg clone https://bitbucket.org/sinbad/ogre/
$ cd ogre
$ mkdir build
$ cd build
# ln -fs /usr/lib/libcurl.so.4 /usr/local/lib/
$ cmake ../
$ make
# make install
