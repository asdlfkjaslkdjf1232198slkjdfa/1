10 “truquillos” para usuarios/as de GNU/Linux – Vol 2.

13 octubre, 2016 En # SYSTEM

  1. Borrado de historial

    export HISTFILESIZE=0
export HISTSIZE=0
export HISTFILE

  2. Filtrar con ngrep peticiones mySQL

    apt-get install ngrep
ngrep -d eth0 -i 'select' port 3306
ngrep -d eth0 -i port 3306

  3. Limpieza de caché Drupal desde la base de datos

    apt-get install drush
drush sql-cli
truncate table cache;
truncate table cache-block;
truncate table cache-bootstrap;
truncate table cache-field;
truncate table cache-filter;
truncate table cache-form;
truncate table cache-image;
truncate table cache-menu;
truncate table cache-page;
truncate table cache-path;
truncate table cache-token;
truncate table cache-update;

  4. Limpiar caché route

    ip route show cache
ip route flush cache

  5. Partir en volúmenes rar

    Por ejemplo en volúmenes de 700MB (más o menos):

    rar a -v700000k volumenresultante.rar /directorio/a/copiar

    Irá partiendo en volúmenes: volumenresultante1.ra, volumenresultante2.rar , …

  6. Crear un php con una cookie de prueba

    <?php
$value='hola es una prueba';
setcookie("testcookie",$value)
setcookie("testcookie",$value, time()+3600);
?>

  7. Bloqueo a ping con iptables

    iptables -A INPUT -i eth0 -P ICMP --icmp-type echo-request -j DROP

  8. Comprimir partiendo en volúmenes con zip

    zip -s 950m -r archivo.zip /ruta/del/directorio

  9. Comprimir y descomprimir con zip

    zip -r algo.zip directorio --> comprimir
unzip algo.zip --> descomprimir

  10. Comprobar cabeceras  web con curl desde la línea de comandos

    curl -k -u usuario:password -I "https://lavidaesasi.com/"
curl  -u usuario:password -I "http://lavidaesasi.com/"
curl -I "http://lavidaesasi.com/"
