10 “truquillos” para usuarios/as de GNU/Linux – Vol 3.

13 octubre, 2016 En # SYSTEM

  1. Obtener el color  (#hex y RGB) de algo que tenemos en pantalla

    apt-get install grabc; grabc

  2. Comprobar cola en exim

    exim -bq | exiqsum

  3. Mostrar porcentaje al importar base de datos mysql

    apt-get install pv
pv bk-thinkbig.sql | mysql -u usuario -pPassword

  4. Cambiar al vuelo puerta de enlace

    route del -net 0.0.0.0 netmask 0.0.0.0 gw 192.168.1.1
route add -net 0.0.0.0 netmask 0.0.0.0 gw 192.168.1.2
route

  5. Configurar un proxy http en la shell

    export http_proxy="http://192.168.1.234:8080"

  6. Alias para conectarse a un servidor windows remoto con rdesktop

    apt-get install rdesktop
alias maquinaremota='rdesktop  -u administrador -p "password" -k es -g 1024x720 -a 16 -z -x l -0 -5 -r sound:off -T "maquina windows" 192.168.1.200'

  7. Dejando un espacio antes de ejecutar una orden en la línea de comandos puede no dejar huella en el history.

  8. Mostrar las vistas en una base de datos

    SHOW FULL TABLES IN nombrebasededatos WHERE TABLE_TYPE LIKE 'VIEW';

  9. Puedes crear nuevos registros con la misma cuenta. Por ejemplo:

    fantaejemplo+1@gmail.com
fantaejemplo+2@gmail.com
fantaejemplo+999@gmail.com
fantaejemplo+foronoseque@gmail.com
fantaejemplo+blablabla@gmail.com

Llegarían todas a: fantaejemplo@gmail.com

  10. Ejecutar un script sql desde la consola mysql para poder ver la salida y que se quede grabada

    script
mysql -u usuario -pPassword basededatos
source script.sql
exit
exit
