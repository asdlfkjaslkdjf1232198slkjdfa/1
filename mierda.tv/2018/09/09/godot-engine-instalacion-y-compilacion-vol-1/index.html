Godot Engine – Instalación y compilación – Vol. 1

9 septiembre, 2018 En # CODE, # GAMES

El año pasado dejé en un refrito algo de info de como instalar Godot y Escoria: https://mierda.tv/2017/08/16/refrito-2-cosas-sobre-las-que-nunca-llegue-a-escribir/

Pero Godot sigue vivo y siguen saliendo mejoras y arreglos. De modo que voy a ver si esta vez puedo meterle caña como para saber como hacer algo decente con Godot. Iré creando pequeños posts que puedan servir a alguien que quiera comenzar como yo.

Descargar y ejecutar Godot rápidamente:

$ wget "https://downloads.tuxfamily.org/godotengine/3.0.6/Godot_v3.0.6-stable_x11.64.zip"
$ unzip Godot_v3.0.6-stable_x11.64.zip 
$ ./Godot_v3.0.6-stable_x11.64

Si deseas compilarlo:

# apt-get install build-essential scons pkg-config libx11-dev libxcursor-dev libxinerama-dev libgl1-mesa-dev libglu-dev libasound2-dev libpulse-dev libfreetype6-dev libssl-dev libudev-dev libxi-dev libxrandr-dev
$ git clone https://github.com/godotengine/godot
$ cd godot
$ scons platform=x11

Y ya. Saludos cordiales.
