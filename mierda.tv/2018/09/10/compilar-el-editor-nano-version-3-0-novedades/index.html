Compilar el editor Nano versión 3.0 – Novedades

10 septiembre, 2018 En # CODE, # SYSTEM

Buenas noticias: https://nano-editor.org/news.php
La versión del editor nano ha alcanzado la 3.0. Entre las mejoras está la velocidad de acceso a los archivos (ahora es mucho más rapido), se puede usar control+suprimir para eliminar la siguiente palabra en un texto, control+shift+suprimir para eliminar la anterior, tiene copiar y pegar y deshacer, etc…

Se puede compilar así:

$ wget https://nano-editor.org/dist/v3/nano-3.0.tar.gz
$ tar xfvz nano-3.0.tar.gz
$ cd nano
$ ./configure
$ make
# make install

No es vi, no es emacs, … pero es un editor que permite editar texto sin tener que hacer un master antes.
