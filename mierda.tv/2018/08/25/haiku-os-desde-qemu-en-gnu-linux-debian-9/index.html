Haiku OS desde Qemu en GNU/Linux Debian 9

25 agosto, 2018 En # SYSTEM

Haiku == El nombre de un sistema operativo. Web oficial: https://www.haiku-os.org/

OS == Operating System, Sistema operativo.

Qemu == Un emulador de procesadores con capacidad de virtualización.

GNU == El proyecto GNU de software libre.

Linux == El núcleo de un sistema operativo.

Debian 9 == La versión 9 de la distro GNU/Linux Debian. Es una distribución muy famosa.

Joe, Es que todo ha de explicarse. No sirve con decirle a la gente “Haiku OS desde Qemu en GNU/Linux Debian 9” para que entiendan que vas a explicar como ejecutar el sistema operativo Haiku en Debian utilizando para ello el emulador Qemu.

Desde este mes de Agosto no suelo explicar demasiado los posts (se explica un poco aquí el motivo: https://mierda.tv/2018/07/30/nuevo-estilo-de-mierda-tv-texto-plano/ ). De modo que vamos ya al turrón y nos dejamos de relleno sin sentido que te haga sentir gilipollas y a mi perder tiempo:

# apt-get install qemu
$ wget "http://ftp.lip6.fr/pub/haiku/releases/r1alpha4.1/haiku-r1alpha4.1-iso.tar.xz"
$ tar xfvJ haiku-r1alpha4.1-iso.tar.xz
$ qemu-img create -f qcow2 haiku.qcow2 8G
$ qemu-system-x86_64 -m 1G -enable-kvm -cdrom haiku-r1alpha4.iso -hda haiku.qcow2 -boot d

Liberar el ratón de la ventana de qemu pulsando control + alt o control + alt + clic izquierdo.

Una vez se instala Haiku se lanza ya así:

$ qemu-system-x86_64 -m 1G -enable-kvm -hda haiku.qcow2

Y poco más que contar.

Saludos cordiales.
