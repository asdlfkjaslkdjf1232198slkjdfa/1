Opendune – Dune II, la batalla por Arrakis en GNU/Linux

3 junio, 2018 En # GAMES

OpenDUNE es una recreación libre del juego “Dune II, La batalla por Arrakis”.

Juego desarrollado originalmente por Westwood Studios y lanzado por Virgin Entertainment.
Open Dune intenta recrear el juego original aplicando tecnología moderna para permitir que se ejecute de forma nativa en la mayoría de los sistemas operativos.

# apt-get install git gcc make libsdl2-dev libasound2-dev cmake g++ libqt5widgets5 qt5-default libsdl2-image-dev libopenlibm-dev libpulse-dev
$ git clone "https://github.com/OpenDUNE/OpenDUNE"
$ cd OpenDUNE/
$ git clone https://github.com/munt/munt
$ cd munt
$ cmake -DCMAKE_BUILD_TYPE:STRING=Release .
$ make
# make install
$ cd ..
$ ./configure LDFLAGS=-lm
$ make
# make install

Con eso ya tenemos compilado e instalado opendune. Para ejecutar el juego hemos de hacerlo escribiendo “opendune” pero … mejor antes bajar los datos del juego.

# mkdir -p /usr/local/opendune
# wget "http://mierda.tv/descargas/opendune.tar.gz"
$ tar xfvz opendune.tar.gz 
# mv opendune /usr/local/

Github de OpenDUNE: https://github.com/OpenDUNE/OpenDUNE
