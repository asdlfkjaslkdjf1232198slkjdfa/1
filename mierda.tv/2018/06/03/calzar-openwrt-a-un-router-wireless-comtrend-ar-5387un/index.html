Calzar OpenWrt a un router wireless Comtrend AR-5387un

3 junio, 2018 En # CODE, # SYSTEM

Este modelo de router lo ofrecía Jazztel en España.
La idea es quitarle el firmware que lleva y ponerle OpenWrt. El proceso es sencillo.

Lo primero es bajarnos el firmware así:

$ wget "https://downloads.openwrt.org/chaos_calmer/15.05/brcm63xx/generic/openwrt-15.05-brcm63xx-generic-AR-5387un-squashfs-cfe.bin"

Lo segundo es apagar el router. Luego resetear metiendo la punta de un clip o algo así en el orificio para resetearlo y dejando este presionado durante más o menos 20 segundos. Mientras se deja presionado se pulsa power para encenderlo.

En cuanto a la forma en la que conectarnos a este router es bueno ponernos de IP: 192.168.1.2 con mascara de red de 24 (255.255.255.0) y puerta de enlace 192.168.1.1.

Si todo va bien al acceder a 192.168.1.1 veremos algo así:

Subimos el firmware que hemos bajado antes y esperamos unos 2 minutos (o 3).

Finalmente recargamos http://192.168.1.1/ en el navegador para colocarle password al usuario root. Y luego ya trastear el router para configurarlo como queramos.

Y eso es todo. Como puede verse este modelo es muy sencillo actualizarle el firmware.
