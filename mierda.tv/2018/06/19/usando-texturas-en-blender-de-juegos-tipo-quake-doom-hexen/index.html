Usando texturas en Blender de juegos tipo Quake, Doom, Hexen, …

19 junio, 2018 En # 3D, # CODE, # GAMES

Necesario:

– 0. Tener blender instalado. ( # apt-get install blender )
– 1. Wad con las texturas del quake2 ( https://www.quaddicted.com/files/wads/Q2Tex.zip )
– 2. Compilar qpakman para extraer las texturas del wad:

$ git clone "https://github.com/bunder/qpakman"
$ cd qpakman
$ cmake .
$ make
# make install

– 3. Descomprimir las texturas usando qpakman

$ unzip Q2Tex.zip
$ qpakman -e q2TEX00.wad )

– 4. Grafx2 para editar las texturas con la paleta indexada correspondiente. ( # apt-get install grafx2 )

– 5. Este archivo puede abrirse con Grafx2 para crear texturas nuevas con la paleta adecuada: quake_palette.zip (el archivo *.pal).

– 6. Utilizar las texturas sobre objetos en blender.

Esto con las texturas originales. Existen packs con las texturas permitiendo una paleta de alta resolución.
