Sc – Hoja de cálculo para línea de comandos

26 junio, 2018 En # CODE

Sc es una de esas herramientas para línea de comandos que molan un montón. Para muchas tareas no es necesaria una hoja de calculo gráfica. Mismamente con emacs en org-mode puedes realizar operaciones o con sc gestionar una hoja de calculo desde la línea de comandos en un terminal.

Me recuerda al visicalc pero guay y para GNU/Linux. Aquí se puede encontrar un manual para aprender a usar SC: http://www.adrianjwells.freeuk.com/minmansc.pdf

Y posiblemente para determinadas tareas esto sea más que suficiente. Todo es cuestión de necesidades y soluciones. Algunas personas necesitan grandes soluciones del mismo modo que para ir a por el pan a la tienda del barrio tienen que coger el coche o un camión en vez de salir andando un minuto y subir el pan a casa.

Como alternativa libre a Excel existen unas cuantas. La más popular es la de la suite de LibreOffice (Calc).
Calc de LibreOffice no obstante es algo que lo mismo se queda grande para mucha gente. Si no te interesa algo así puedes probar con gnumeric por ejemplo que es algo más sencillo y consumirá muchos menos recursos.

Si aún así tampoco quieres algo de eso lo mismo sc es lo que te viene bien. Consumirá pocos recursos y podrás realizar lo mismo que otra gente.

Si se siente la necesidad de crear gráficas no es complicado enchufarle a esto gnuplot.

Para instalar esto en debian se puede por ejemplo así:

# apt-get install sc
