OpenRA – Command & Conquer: Red Alert en GNU/Linux

4 junio, 2018 En # GAMES

Con OpenRA es posible jugar entre otros juegos al “Command & Conquer: Red Alert”. Es necesario disponer del juego original para lo que vienen siendo los gráficos, etc …

Instalar OpenRA en Debian 9 es muy sencillo. Se puede hacer por ejemplo así:

$ wget "https://github.com/OpenRA/OpenRA/releases/download/release-20180307/openra_release.20180307_all.deb"
# dpkg -i openra_release.20180307_all.deb 
# apt --fix-broken install
# dpkg -i openra_release.20180307_all.deb

Para ejecutarlo y jugarlo se puede escribiendo:

$ openra-ra

No obstante necesitamos los cdroms del juego original. Aquí pueden bajarse:

$ wget "https://mierda.tv/descargas/CD1_ALLIED_DISC.ISO"
$ wget "https://mierda.tv/descargas/CD2_SOVIET_DISC.ISO"

Hemos de montarlos. Por ejemplo así:

# mount -o loop CD1_ALLIED_DISC.ISO /media/cdrom0

Y entonces si que podemos usar ya openra-ra para escanear el contenido. Se puede desmontar con “umount /media/cdrom0” y montar el otro CD.

Una vez realizado ya no necesitaremos las isos de los cdrom originales.

NOTA: Presiona control y con el ratón ruleta del medio podrás hacer zoom. Se verán los monigotes algo pixelados pero … eso forma parte del encanto.

WEB: http://www.openra.net/
