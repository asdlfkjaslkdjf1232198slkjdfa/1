Bios de miles de maquinas y Kaillera server mame en GNU/Linux

31 julio, 2018 En # GAMES, # SYSTEM

Jugar y disfrutar con la BIOS de un montonazo de maquinas es una bonita experiencia:

# apt-get update
# apt-get install mame gnome-video-arcade mame-tools mame-extra mame-data mame-doc
$ wget "https://mierda.tv/descargas/bios.zip"
$ unzip bios.zip
$ mame -verifyroms roms/
$ mame -rp roms/

Kaillera server en GNU/Linux:

$ wget http://kaillera.abime.net/kaillerasrv-0.86-linux.tgz
$ tar xfvz kaillerasrv-0.86-linux.tgz 
$ cd kaillerasrv-0.86/
$ ./kaillerasrv
