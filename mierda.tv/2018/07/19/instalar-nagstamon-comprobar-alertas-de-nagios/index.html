Instalar nagstamon – Comprobar alertas de nagios

19 julio, 2018 En # CODE, # SYSTEM

En GNU/Linux usar nagstamon es buena cosa si necesitas gestionar varios nagios. Esto permite disponer de diferentes alertas que vienen de diferentes proyectos/clientes/… y en resumen de diferentes servidores nagios que previamente configuras.

En debian puedes encontrar nagstamon pero no la última versión. Si deseas la última versiones puedes hacer lo siguiente:

# apt-get install python-configparser python3-keyring python-keyring python3-psutil python3-pyqt5 python3-pyqt5.qtsvg python3-pyqt5.qtmultimedia python3-bs4
# apt-get install python3-requests-kerberos
$ git clone "https://github.com/HenriWahl/Nagstamon"
$ cd Nagstamon/
$ python setup.py build
# python setup.py install

Con esta herramienta tendrás a mano y mucho mejor que en web (con notificación con sonido, etc…) la posibilidad de saber que webs, servicios, maquinas, … están caidos, sobrecargados, …
