Netsurf – Navegando sin javascript por defecto

14 abril, 2018 En # CODE

Debian 9 trae en los repositorios stable la versión de netsurf 3.6. Funciona bien de modo que tampoco es tan vital compilar la 3.7.

Para instalar netsurf en debian:

# apt-get install netsurf

Netsurf es un navegador que vendrá con javascript deshabilitado por defecto. Sirve para navegar en webs aunque cierto es que no veremos muy bien muchas de ellas. Si necesitamos algo más que leer información para eso mejor otros navegadores más modernos.

Lo interesante de netsurf es que su engine está escrito en C y eso lo hace bastante fácil de portar a otras plataformas. Lo malo es que no tiene un soporte fino filipino de css y html pero sirve para navegar con un poco más de control.

Si se desea compilar la 3.7 lo primero es instalar algunas dependencias:

# apt-get install libcurl4-openssl-dev libpng-dev gperf make gcc libhpdf-dev

Los siguientes pasos del tirón para compilarlo:

$ wget http://download.netsurf-browser.org/netsurf/releases/source-full/netsurf-all-3.7.tar.gz
$ tar xfvz netsurf-all-3.7.tar.gz
$ cd netsurf-all-3.7/netsurf/docs/
$ cp -pRv env.sh ../../
$ cd ../..
$ source env.sh
$ ns-package-install
$ ns-clone 
$ ns-pull-install
$ rm env.sh
$ cd ~/dev-netsurf/workspace
$ source env.sh
$ cd netsurf/
$ make
# cp -pRv nsgtk /usr/bin/netsurf
