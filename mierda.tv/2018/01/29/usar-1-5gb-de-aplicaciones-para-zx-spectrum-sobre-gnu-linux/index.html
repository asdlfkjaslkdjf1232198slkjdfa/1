Usar 1.5Gb de aplicaciones para ZX Spectrum sobre GNU/Linux

29 enero, 2018 En # GAMES

Antes de nada relájate amigo. Vamos a descargar un zip de 1.5Gb de un montón de aplicaciones para ZX Spectrum y un emulador llamado fuse. Al mismo tiempo para que todo funcione veremos como montar el tinglado de modo que si dispones de unos minutos podrás realizar rápidamente todo lo necesario y luego ya disfrutar de la colección de cintas.

# Dejémonos de tonterías y vamos al grano

# apt-get install fuse-emulator-common fuse-emulator-utils spectrum-roms

Con eso instalamos el software necesario.

La lista de ROMs incluidas en el paquete spectrum-rooms es esta:

48.rom – Original 48K ROM
128-0.rom – 128K ROM 0 (128K editor and menu)
128-1.rom – 128K ROM 1 (48K BASIC)
128-spanish-0.rom – Spanish 128K ROM 0 (128K editor & menu)
128-spanish-1.rom – Spanish 128K ROM 1 (48K BASIC)
if1-2.rom – Interface 1 v2 ROM
plus2-0.rom – +2 ROM 0 (128K editor and menu)
plus2-1.rom – +2 ROM 1 (48K BASIC)
plus3-40-0.rom – +2A/+3 v4.0 ROM 0 (128K editor)
plus3-40-1.rom – +2A/+3 v4.0 ROM 1 (128K syntax checker)
plus3-40-2.rom – +2A/+3 v4.0 ROM 2 (+3DOS)
plus3-40-3.rom – +2A/+3 v4.0 ROM 3 (48K BASIC)
tc2048.rom – Original 48K ROM modified for the Timex TC2048

Las ROMs se encontrarán en /usr/share/spectrum-roms/

# Montar el tinglado para tenerlo todo a mano y sencillo

Creamos un directorio en nuestro home llamado por ejemplo fuse y dentro otros 2 llamados roms y tapes:

$ mkdir fuse;cd fuse;mkdir roms tapes

Copiamos las roms al directorio roms recién creado así:

$ cp -pRv /usr/share/spectrum-roms/* roms/

Ahora es cuando nos bajamos un montón de cintas del proyecto TOSEC para zxspectrum. Pueden bajarse desde archive.org así:

$ wget "https://archive.org/download/Sinclair_ZX_Spectrum_TOSEC_2012_04_23/Sinclair_ZX_Spectrum_TOSEC_2012_04_23.zip"
$ unzip Sinclair_ZX_Spectrum_TOSEC_2012_04_23.zip
$ rm -rf tapes/ 
$ mv "Sinclair ZX Spectrum [TOSEC]" tapes
$ rm -rf Sinclair_ZX_Spectrum_TOSEC_2012_04_23.zip

Se trata de un zip de 1.5 Gb repleto de cintas de todo tipo. Desde aplicaciones a juegos o revistas electrónicas. Puedes comprobar su contenido aquí: http://ia601601.us.archive.org/zipview.php?zip=/11/items/Sinclair_ZX_Spectrum_TOSEC_2012_04_23/Sinclair_ZX_Spectrum_TOSEC_2012_04_23.zip

Relájate un poco de todos modos ya que tardará algo en descomprimir el zip.

Ya tenemos cintas y tenemos las roms de los ordenadores spectrum. Es el momento de ejecutar fuse y disfrutar.

Ahora ya solo nos queda en el emulador fuse seleccionar la rom adecuada (options –> select ROMs –> Spectrum 48K …) y luego la cinta adecuada.
Si la cosa se ve pequeña encontraremos la opción de mostrar la ventanita al doble o el triple de su tamaño.

Encontraremos cintas para por ejemplo organizar nuestra agenda de contactos. Algo que puede seguir siendo de utilidad al día de hoy por ejemplo:
