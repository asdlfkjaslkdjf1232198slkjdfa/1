One liner – Bajar y escuchar módulos musicales aleatoriamente

12 enero, 2018 En # MUSIC

Es importante disponer de 2 programas para que esto chute bien: wget y xmp .

En GNU/Linux Debian 9 se pueden bajar ambos así:

# apt-get install wget xmp

El One liner es este:

$ wget -q $(wget -q "https://modarchive.org/index.php?request=view_random" -O - | grep "standard-link" | grep "Download" | cut -d "\"" -f 2) -O random.module; xmp random.module

Si deseamos añadirlo a un script llamado “musica” podemos hacerlo con este contenido:

#!/bin/bash
wget -q $(wget -q "https://modarchive.org/index.php?request=view_random" -O - | grep "standard-link" | grep "Download" | cut -d "\"" -f 2) -O /tmp/random.module; xmp /tmp/random.module

Guardamos el archivo como: musica

Le damos permisos de ejecución y lo copiamos a por ejemplo /usr/local/bin/

$ chmod +x musica
# cp -pRv musica /usr/local/bin/

Ahora cada vez que escribamos música en la consola nos sacará un modulo de sonido diferente (que podemos cortar con control+c si no nos mola mucho).
