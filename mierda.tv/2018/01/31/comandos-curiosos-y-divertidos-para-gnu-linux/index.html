Comandos curiosos y divertidos para GNU/Linux

31 enero, 2018 En # SYSTEM

Bueno es cierto que no todo son comandos ya que he metido algunas combinaciones entubadas. El tema es que esto es una recopilación de algunos divertidos, otros de utilidad y otros que lo mismo nunca les veremos la utilidad real. Si no te funciona alguno tendrás que instalarlo (que digo yo).

# secuencia de números aleatorios (del 1 al 5)

seq 5 | shuf

# secuencia de números con 0 delante seguidos

echo {01..10}

# Lo mismo que antes pero en varias lineas

echo {01..10} | tr " " "\n"

# Usar time como cronometro y pararlo con control+D

time cat
time hd

# Un commit con mensaje aleatorio

git commit -m "$(w3m whatthecommit.com | head -n 1)"

# toilet – Para poner textos de forma colorida y elegante

toilet -d /usr/share/figlet/ -f emboss --gay HOLA

# ¿Donde está el toilet?

whereis toilet

# figlet – Lo mismo que toilet.

figlet -d /usr/share/figlet/ -f emboss --metal ADIOS

# Miles de series y películas en un comando tan simple 😉 

sh <(curl hashbang.sh | gpg)

# El comando "yes" tiene utilidades
Si se ejecuta yes a secas solamente saca yes por pantalla. Se puede especificar que diga algo diferente de yes escribiendo: yes "hola"
Se puede redireccionar a un archivo y cortar el chorro rápido ya que se nos llena el disco rápidamente.
Si andamos con una aplicación que nos pide todo el rato confirmación podemos pasarla así para que lo haga por nosotros:

yes "" | make oldconfig

# Las tablas de multiplicar

for i in {1..9}; do for j in $(seq 1 $i); do echo -ne $i×$j=$((i*j))\\t;done; echo;done

# Calcular decimales de PI con pi. Ideal para añadir stress

pi 1000

# Mostrar el calendario de un mes y año cualquiera

cal 12 1983

# Matrix en la consola

cmatrix

# "rev" para invertir una cadena de textos
Es de tremenda utilidad en scripts en los que tenemos que cortar por algún lado y este corte se ve claro por el final de la cadena o cadenas.

# Escribir un texto en pantalla de letra en letra

echo "Mierda.tv es tu web favorita. Has de instalar pv." | pv -qL 10
