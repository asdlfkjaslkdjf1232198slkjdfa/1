Instalar Docker sobre debian 9 stretch

10 enero, 2018 En # SYSTEM

En diciembre del 2016 publicamos un post sobre como instalar docker sobre debian 8. Este va sobre como instalar docker sobre debian 9.

Comprobamos la versión del Kernel así:

$ uname -r

Tendría que ser superior a la 3.10
Si comprobamos con -a veremos que la versión de debian tiene que ser de 64.
Comprobamos que estamos en debian 9 por ejemplo así:

$ cat /etc/issue

Añadimos el repositorio de docker en /etc/apt/sources.list

deb [arch=amd64] https://apt.dockerproject.org/repo debian-stretch main

Instalamos algunas dependencias:

# apt-get install apt-transport-https ca-certificates gnupg2

Luego importamos clave del repo y lo actualizamos así:

# wget -q "https://apt.dockerproject.org/gpg" -O - | apt-key add
# apt-get update

Y finalmente simplemente instalamos ya docker-engine así:

# apt-cache policy docker-engine
# apt-get install docker-engine

Si debian 9 está en un contenedor LXC no necesitaremos lo siguiente. Si es tu maquina de tu PC pues lo mismo requieres añadir esto a /etc/default/grub :

GRUB_CMDLINE_LINUX_DEFAULT="cgroup_enable=memory swapaccount=1"

Tras guardar ejecutamos:

# update-grub

Reiniciamos el sistema y ya está listo para usarse si todo va bien.

Si queremos añadir un usuario para usar docker en vez de usar root lo podemos hacer así (el ejemplo es el usuario fanta):

# groupadd docker
# gpasswd -a fanta docker
# /etc/init.d/docker restart
