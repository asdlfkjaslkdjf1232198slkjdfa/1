Efectos para la voz con pulseeffects

25 mayo, 2018 En # CODE, # MUSIC

Es posible instalar pulseeffects en Debian 9 stable utilizando flatpak ya que compilarlo nos pedirá versiones muy nuevas de librerías y eso nos supondrá bastantes problemas posiblemente.

Lo interesante de este software es que si dispones de un micro de condensador usb (uno al estilo samson meteor por ejemplo ya que no requiere que lo pases por una mesa que tenga alimentación phantom -48v) o cualquier micro de mierda, … pues como que puedes añadirle efectos a por ejemplo webrtc, es decir, si por ejemplo tienes una videoconferencia mediante el navegador utilizando webrtc podrás al vuelo añadir efectos a tu voz. Eso supone que puedes añadirle compressor por ejemplo.

De modo que flatpak es buena opción en este caso. La forma de instalar pulseefects es esta:

# apt install flatpak
# apt install gnome-software-plugin-flatpak
# flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
# reboot
# flatpak install flathub com.github.wwmm.pulseeffects

Y no solamente al micro. Podemos abrir el vlc una peli y añadirle a la salida efectos para que el audio quede algo más decente o adaptado a nuestras necesidades con los recursos de los que disponemos.

No lo he probado pero supongo que con mumble funcionará. Lo bonito es que puedes crearte perfiles y nombrarlos para cada aplicación.

Por otro lado este tipo de software nos evita en cierto modo tener que tener un equipo de voces con fx, es decir, podemos trabajar directamente con micros de condensador (como el nombrado anteriormente o similares) sin tener una mesa de mezclas con alimentación y sin tener en hardware mesa de efectos. Todo directamente a un portatil y a funcionar.

No será lo mismo que trabajar esas cosas a mano con hardware de verdad pero … funcionar funciona :).
