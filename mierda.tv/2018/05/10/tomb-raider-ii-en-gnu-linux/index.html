Tomb Raider II en GNU/Linux

10 mayo, 2018 En # GAMES

Es posible jugar a Tomb Raider II en GNU/Linux de varias formas. Una de ellas es con wine (y es la que se explica en este post) y otra es con OpenTomb.

Sea como sea se necesitan los datos originales del juego. Se puede bajar listo para jugar el Tomb Raider II desde aquí: https://mierda.tv/descargas/Tomb2.tar.gz

Es importante ejecutar winecfg antes y elegir algo así como windows xp como el sistema que utilizarás cuando ejecutes algo con wine.

En winecfg se puede elegir donde vas a montar el cdrom como unidad d por ejemplo.

El iso que viene se ha de montar para que podamos ejecutar el juego tal y como se ve en la siguiente captura:

# mount -o loop Tomb2.iso /media/cdrom
$ wine Tomb2.exe
