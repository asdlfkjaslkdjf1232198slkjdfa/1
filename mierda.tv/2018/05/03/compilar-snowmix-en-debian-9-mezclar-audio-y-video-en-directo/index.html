Compilar Snowmix en Debian 9 – Mezclar audio y vídeo en directo

3 mayo, 2018 En # MUSIC

Snowmix es un programa que podría servirnos para mezclar video y audio en directo. OBS también nos permite esto pero supuestamente snowmix permite bastante trasteo (yo aún no lo he probado bien ya que requiere de leer mucho). En este post simplemente se relata como puede compilarse en Debian 9.

Descargamos el código: https://sourceforge.net/projects/snowmix/files/latest/download

Lo podemos dejar en nuestro home por ejemplo o en un directorio llamado tmp dentro de nuestro home. Eso es lo de menos.
El archivo tendrá un nombre similar o igual a “Snowmix-0.5.1.tar.gz” y ocupará unos 59Mb.

Antes de nada vamos a instalar las dependencias:

# dpkg --add-architecture i386
# apt-get update
# apt-get install build-essential automake autoconf libtool freeglut3-dev
# apt-get install g++ pkg-config libsdl1.2-dev gstreamer1.0-tools libosmesa6-dev 
# apt-get install libpango1.0-dev tcl-dev tcl tk bwidget bc
# apt-get install faac libfaac-dev libfaac0

Ahora vamos a descomprimir el tar.gz del código fuente y vamos a acceder al directorio que se creará:

$ tar xfvz Snowmix-0.5.1.tar.gz 
$ cd Snowmix-0.5.1

Ejecutamos bootstrap pero antes metiendo nuestro usuario a sudo.

# apt-get install sudo
# adduser nombreusuarioexistentequevamosausar sudo
$ ./bootstrap

Con las dependencias que hemos instalado antes esto tendría que tardar poco. Nos pedirá la password del usuario que andamos usando y que está en sudo.
Es posible que tengamos que ejecutar bootstrap varias veces.

Si nos peta con libpng hacemos esto pero nunca instalar libpng-dev que es otra versión:

$ wget "http://ftp.us.debian.org/debian/pool/main/libp/libpng/libpng12-dev_1.2.50-2+deb8u3_i386.deb"
# dpkg -i libpng12-dev_1.2.50-2+deb8u3_i386.deb

Ahora si todo va bien vamos a compilar Snowmix:

$ ./configure 
$ make
# make install

Añadimos a .bashrc del home de nuestro usuario:

export SNOWMIX=/usr/local/lib/Snowmix-0.5.1

Ahora ya podemos probar las demos y trastear con snowmix.

snowmix_demo
