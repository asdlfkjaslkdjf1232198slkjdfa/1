Descomprimir ISO con 7z en GNU/Linux

3 mayo, 2018 En # SYSTEM

Algunas veces en algunas distros mínimas nos será complicado usar el comando mount loop para montar una iso de un DVD/CD. Si nuestra intención es solamente acceder a los archivos de la ISO y no queremos instalar mil dependencias (por ejemplo si andamos en un server) es posible utilizar 7z para esto. Por ejemplo así:

# apt-get install p7zip-full
$ 7z x quake4.iso

Eso descomprimirá la ISO y ya podremos copiar los archivos que necesitemos de su interior.
