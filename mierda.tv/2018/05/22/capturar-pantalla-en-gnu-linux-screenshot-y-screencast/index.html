Capturar Pantalla en GNU/Linux – Screenshot y Screencast

22 mayo, 2018 En # CODE

Varios programas para capturar imágenes estáticas de la pantalla, capturar vídeo, gifs de algo en pantalla, … es lo que viene a relatarse en este post.

# Peek

Peek mola un montón. No viene en repos de Debian por ejemplo pero no es difícil instalarlo.
La finalidad de Peek es la de crear un gif, un apng (un png animado), un vídeo, … de una parte de la pantalla que deseas.

Un ejemplo de gif:

La forma de instalarlo en Debian puede ser esta:

# apt install cmake valac libgtk-3-dev libkeybinder-3.0-dev libxml2-utils gettext txt2man
$ git clone https://github.com/phw/peek.git
$ mkdir peek/build
$ cd peek/build
$ cmake -DCMAKE_INSTALL_PREFIX=/usr -DGSETTINGS_COMPILE=OFF ..
$ make package
# dpkg -i peek-*-Linux.deb

# Scrot

Scrot recuerda a la palabra “escroto” de algún modo. A mi me sirve eso para acordarme del nombre aunque no tengan nada que ver una cosa con la otra.
Ese software para línea de comandos nos permitirá capturar la pantalla, es decir, lo que viene siendo hacer un screenshot, un pantallazo, …

Para sacar una captura:

$ scrot

Para sacar una captura de una ventana que seleccionemos con el ratón:

$ scrot -s

Para sacar una captura con un delay de x segundos (para que de tiempo a por ejemplo posicionar el ratón en algún lado o lo que sea):

$ scrot -d 3

Está en repos de Debian.

# Shutter

Shutter es un programa gráfico. Nos permitirá sacar capturas tanto con delay, como de una zona especifica de la pantalla, de un escritorio, …
Aparte de las captpuras es interesante ya que permite gestionar las capturas e incluso subirlas rápidamente a algún servicio de subida de fotos y compartir la url con alguien.

Shutter es muy recomendable si no tienes problemas con que ocupe cierto espacio. No es un software minimalista pero si que es muy interesante por las opciones que ofrece todo en 1.

Está en repos de Debian.

# Vokoscreen

Vokoscreen

Si no precisas a bruce willis a tope con la cope en tu pantalla lo mismo vokoscreen te sirve. Con eso lo que quiero decir es que para grabar por ejemplo un juego de acción no es el mejor software y que incluso es mucho mejor capturar esas cosas con harware. Existe mejor software que Vokoscreen para capturar partidas de juegos pero … pero … digamos que vokoscreen cumple muy bien para muchas otras cosas.

Para todo lo que no sean juegos o animaciones muy rápidas vokoscreen sirve. Puedes capturar webcam al mismo tiempo que la pantalla, capturar una zona determinada, …

Está en repos de Debian.

# Simplescreenrecorder

Simplescreenrecorder a mi me gusta. No permite grabar cámara al mismo tiempo que se captura la pantalla pero si que permite afinarlo bien para capturar juegos como Quake 2 por ejemplo. A diferencia de Vokoscreen este si se puede afinar un poquito para capturar algo que precise de movimientos más rápidos.

El tema de perfiles es muy buena cosa. Puedes crearte perfiles para por ejemplo capturar el juego Larry 7, otro para el Quake2, etc… y afinar tema de tasa de fotogramas, codecs de vídeo/audio, …

Está en los repos de Debian.

Existen muchos más pero estos son unos de los que yo utilizo aparte de OBS-studio. ¿Conoces alguno más?
